Our original goal was to scrape a cat fact from the cat facts API and display that on the screen connected to Allison's ESP32 when a button was pressed. However, the cat facts API required authentication, so we switched to a random facts API with no required authentication instead. We were set to have one ESP32 send a signal to the other, which would activate a screen to display a random fact.
